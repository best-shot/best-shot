This tool includes the following packages:

- case-sensitive-paths-webpack-plugin
- clean-webpack-plugin
- copy-webpack
- terser-webpack-plugin
- webpack-chain

## Installation

```bash
npm install @best-shot/core --save-dev
```

## Usage

```mjs
import { BestShot } from '@best-shot/core';

new BestShot().setup({
  presets: ['babel'],
  mode: 'production',
  config: {
    polyfill: {
      usage: 'global'
    }
  }
});

new BestShot().setup({
  presets: ['style'],
  mode: 'development',
  watch: true
});
```
