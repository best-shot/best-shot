## Installation

```bash
npm install @best-shot/preset-serve --save-dev
```

## Usage

```mjs
// example: .best-shot/config.mjs
export default {
  presets: ['serve'],
  experiments: {
    lazyCompilation: false // true by default
  },
  devServer: {
    // ...
  }
};
```
